# Google ADK Multi-Agent Workflow System

A production-ready multi-agent AI system built with **Google Agent Development Kit (ADK)** that demonstrates collaborative agents working together to research, analyze, and produce content on any topic.

## Features

- **Multi-Agent Orchestration**: Intelligent routing of tasks between specialized agents
- **Web Research**: Real-time web search and data gathering
- **Text Analysis**: Readability metrics, keyword extraction, and sentiment analysis
- **Content Generation**: Professional reports, emails, and executive summaries
- **Google ADK Native**: Built with Google's Agent Development Kit for seamless integration

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Framework | Google Agent Development Kit (ADK) |
| LLM | Gemini 2.0 Flash |
| Language | Python 3.11+ |
| Tools | Custom function tools with type safety |

## Quick Start

```bash
git clone https://github.com/Nagavenkatasai7/google-adk-multi-agent-workflow.git
cd google-adk-multi-agent-workflow
pip install -r requirements.txt
cp .env.example .env
python tests/test_tools.py
python main.py
```
